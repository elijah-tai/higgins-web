Higgins

Simple tool to help delegate and track the completion of tasks.


Create a group

Have a presentation you'd like to finish early? A lab group that you'd like to keep organized? Housemates to keep accountable? Create a group for anyone and everyone.

Add group members

Higgins uses text messages to deliver upcoming task notifications. Group members can respond to the text to let others know they've accepted the task.

Assign a task

Higgins will send a reminder on the set date/time to notify the assignee that their task is due, and will wait for an update from the assignee on whether the task has been completed or not.

Profit!

Create a group, add members, and assign tasks. Get things done. Repeat.
